# صدقة جارية — حسين أحمد بيومي

A memorial website for remembrance and continuous charity through dhikr (Islamic remembrance).

## Features
- Islamic dhikr counter with progress tracking
- Quranic verses and duas
- Counter persistence using browser storage
- Beautiful Arabic typography
- Dark theme design

## Deployment

### Vercel (Recommended)
1. Go to [vercel.com](https://vercel.com)
2. Click "New Project"
3. Import your GitHub repository
4. Deploy (no build steps needed)

### GitHub Pages
1. Go to repository settings
2. Enable GitHub Pages from the `main` branch
3. Your site will be live at `https://ahmedabora-dev.github.io/hussein-sadaqa`

### Other Platforms
- Netlify: Similar to Vercel
- Render: Connect GitHub and deploy
- Cloudflare Pages: Connect GitHub and deploy
